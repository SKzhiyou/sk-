# 我的聊天礼物网站

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个带有实时聊天室和礼物发送功能的网站

## 功能特性
- 💬 实时聊天功能
- 🎁 发送多种礼物
- 📱 响应式设计
- ⚡ 轻量高效

## 快速开始
1. 注册LeanCloud账号
2. 创建应用，获取APP ID和KEY
3. 替换`app.js`中的配置
```javascript
const APP_ID = "你的APP_ID";
const APP_KEY = "你的APP_KEY";
```

## 部署方式
推荐使用：
- GitHub Pages
- Vercel
- 腾讯云Webify

## 技术栈
- LeanCloud (后端服务)
- HTML/CSS/JavaScript (前端)
